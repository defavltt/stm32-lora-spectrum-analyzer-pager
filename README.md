# STM32 based LoRa spectrum analyzer & pager

<!-- <img src="images/DSC_1560.JPG" width="800"> -->
![](images/DSC_1560.JPG)

## Features

 - Sending and receiving messages through LoRa
 
 - Configuring most of the RFM95's settings
 
 - Spectrum analyzer
 
 - Direct view of RFM95's registers
 
 - Battery measurement
 
 - Backlight
 
 - Vibration
 
 - Sleep modes
